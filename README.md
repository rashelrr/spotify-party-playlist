# Description
A web app that uses the Spotify Web API to generate a playlist based on my top X tracks!

Given the recent Spotify API changes (March 2026), the /recommendations endpoint and genre data is no longer available (deprecated).
https://developer.spotify.com/blog/2026-02-06-update-on-developer-access-and-platform-security

## Installation and Usage
Use pip3 to install necessary dependencies:
```bash
pip3 install -r requirements.txt
```

Run via:
```bash
python3 main.py
```
